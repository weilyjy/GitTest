一、首页
1、导航栏：
	    通过ajax从josn中取出数据添加到导航栏，到鼠标hover导航栏上对应的商品名
	时会加载出不同的商品。例如，到hover到魅族手机时，会出现不同的魅族手机商品，且商品的
	出现伴有动画效果，先从左向右划出，最后在中间位置停止，到鼠标hover到二级菜单中的商品时，
	被hover上去的商品会变亮，其他的商品会变暗，当点击鼠标指定的商品时，会跳转到相应的详情页。
2、侧边栏：
	   通过ajax从josn中取出数据添加到导航栏，到鼠标hover导航栏上对应的商品名
	时会加载出不同的商品。当点击鼠标指定的商品时，会跳转到相应的详情页。
3、轮播图：
	  轮播图的效果是淡入淡出的效果，使用定时器，让6张图片不停的变换，在图片的底部有一组小图标，当hover上去
	不同的小图标的时候会显示不同的图片，在轮播图的中间有两个左右的箭头，点击左箭头图片向前淡入，点击右箭头
	图片向后淡入。
4.商品列表：
	首页上面的商品都是通过ajax获取且可以被点击，点击后跳转到相对应的商品详情页。
二、二级页面
	点击首页上面的一级导航栏会跳转到二级页面商品列表页。二级页面的所有数据都是通过ajax获取。
三、商品详情页：
	点击首页中的商品会跳转到相应的商品详情页，且可以使用放大镜查看商品。
四、购物车界面
	点击详情页中的加入购物车按钮，商品就会被cookie缓存，在购物车界面通过从cookie中取出商品id即可加载出
加入购物车的商品，显示商品的详情，点击X后会弹出提示框提示是否删除该商品，点击确定即可删除，否则就不会被删除。
五、注册登录	
	 注册页面的js是通过单例来写的，且对用户的手机号进行了正则表达式校验。校验成功后存入cookie中。
     登录，首先从cookie中取出注册成功的用户id,和当前输入的进行比对，比对成功，跳转到首页，且在首页显示
     登录成功的提示框，提示框会在1s后自动消失
说明：css是使用scss来写的，使用gulp，使用模块化开发。
